# light-brain
光脑方案是一个生长式神经符号融合通用智能架构。它模拟人脑的分区协作机制，将感知、情感、记忆、推理、语言生成等认知功能拆分为六个独立模块（丘脑、前额叶、杏仁核、小脑、海马体、布罗卡区），通过标准化接口协同工作。  与传统大语言模型不同，光脑方案的核心特征包括：生长而非训练——知识体现为物理性的神经连接生长；符号与神经融合——符号层保障逻辑严密，神经层负责流畅生成；可解释与可编辑——知识以结构化形式存储，可被人类直接阅读和修改；可控的创造力——三旋钮联动回路使创造性成为可调节的连续变量。  光脑方案提供完整的中英双版本代码骨架、配置文件、种子知识库和全套设计文档。当前版本为理论架构完整版，代码开源，供研究社区验证与扩展。
